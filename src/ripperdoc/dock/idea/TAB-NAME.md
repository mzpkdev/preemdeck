# TAB: name this tab by its work

Keep this JetBrains terminal tab labeled with the task in front of you, so a glance across tabs reads the work.

- Choose a 1 to 2 word, lowercase, kebab-case slug for the current task (`auth-retry`, `tab-naming`, `flaky-ci`).
- Apply it by running
  `$HOME/.preemdeck/preemdeck-runtime "$HOME/.preemdeck/src/ripperdoc/dock/idea/toolbox/rename-tab.ts" <slug>`.
- Only (re)name when the current tab name no longer fits the work; if it already reflects what you are doing, do
  nothing, and never announce the rename.
